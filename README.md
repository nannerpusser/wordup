##Wordament Solver App
Description
This Wordament Solver app allows users to input a 4x4 board of letters and find valid words from the board based on a provided dictionary.

Features
Input a 4x4 board of letters
Solve the Wordament board to find valid words
Clear the board entries
Error message for incomplete board entries
Dark mode appearance
Installation
Clone the repository
Install the required dependencies
Copy
Insert
pip install customtkinter
Run the ws.py file
Usage
Launch the application by running the ws.py file
Fill in the 4x4 board with letters
Click the "Solve" button to find valid words
Click the "Clear" button to clear the board entries
